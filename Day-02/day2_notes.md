## 📚 Concepts Learned
### 1. Variables
         Variables are defined as a named memory locaton that's used for storing values of different data types.
          **Example**
              ```Python
                marks = 75
                #Deaclared and initialized at the same time.```
          - They are initialized at the time of decelaration.
          **Example**
              ```Python
                name = 'Ali'```
          - It's value can be changed during execution.
          **Example**
              ```Python
                age = 17
                print("My age is",age) # My age is 17```
          - It can be accessed by it's name.
          **Example**
              ```Python
                name = "Ali"
                print("I'm "name) # I'm Ali
                 
          - Keywords can't be used as the name of variable.
          **Example**
              ```Python
                True = 2 + 2
               # SyntaxError: cannot assign to True```
