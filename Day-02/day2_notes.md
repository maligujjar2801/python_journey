# 📚 Concepts Learned

## 1. Variables

Variables are defined as named memory locations that are used for storing values of different data types.

### Example

```python
marks = 75
# Declared and initialized at the same time.
```

- They are initialized at the time of declaration.

### Example

```python
name = "Ali"
```

- Their value can be changed during execution.

### Example

```python
age = 17
print("My age is", age)  # My age is 17
```

- They can be accessed by their names.

### Example

```python
name = "Ali"
print("I'm", name)  # I'm Ali
```

- Keywords can't be used as variable names.

### Example

```python
True = 2 + 2

# SyntaxError: cannot assign to True
```

## 2. Data Types

These are the types of data that can be stored in variables.

### Integer

It is a data type in Python that is used to store whole numbers in variables.

#### Example

```python
age = 17
```

---

### Floating Point

It is used to store decimal values in variables.

#### Example

```python
temp = 35.4
```

---

### String

It is used to store text messages in variables.

#### Example

```python
name = "Ali"
```

---

### Boolean

It only stores two kinds of values: `True` or `False`.

#### Example

```python
is_raining = True
```

## 3. Operators

Operators are used to perform operations like addition, subtraction, multiplication, etc.

### Arithmetic Operators

#### Addition (`+`)

The `+` operator is used to add two values.

```python
a = 3
b = 4

sum = a + b  # 7
```

---

#### Subtraction (`-`)

The `-` operator is used to subtract two numeric values.

```python
a = 3
b = 4

difference = a - b  # -1
```

---

#### Multiplication (`*`)

The `*` operator is used to perform multiplication.

```python
a = 6
b = 5

product = a * b  # 30
```

---

#### Division (`/`)

The `/` operator is used to perform division.

```python
a = 25
b = 5

division = a / b  # 5.0
```

---

#### Modulus (`%`)

The `%` operator is used to find the remainder after dividing two numbers.

```python
a = 31
b = 3

remainder = a % b  # 1
```

---

#### Exponentiation (`**`)

The `**` operator is used to raise the first value to the power of the second.

```python
a = 5
b = 2

print(a ** b)  # 25
```

---

#### Floor Division (`//`)

The `//` operator is used to divide two numbers and return the largest integer value (rounding down).

```python
a = 5
b = 2

print(a // b)  # 2
```
