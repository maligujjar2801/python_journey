## 1. What is Python?

Python is a high-level, interpreted, and easy-to-read programming language. It is widely used for web development, automation, data science, artificial intelligence, machine learning, and many other fields.

---

## 2. Why is Python interpreted?

Python is called an interpreted language because its code is executed by the Python interpreter line by line instead of being compiled into machine code before execution. This makes it easier to test and debug programs.

---

## 3. Difference between int and float

| int | float |
|------|--------|
| Stores whole numbers | Stores numbers with decimal points |
| Example: `10` | Example: `10.5` |

Examples:

```python
age = 17        # int
height = 5.9    # float
```

---

## 4. Difference between list and tuple

### List
- Mutable (can be changed)
- Uses square brackets `[]`

Example:

```python
fruits = ["Apple", "Banana", "Orange"]
```

### Tuple
- Immutable (cannot be changed after creation)
- Uses parentheses `()`

Example:

```python
coordinates = (10, 20)
```

---

## 5. What is a variable?

A variable is a named location in memory used to store data. Variables make it possible to save and use values later in a program.

Example:

```python
name = "Ali"
age = 17
```

---

## 6. What is a string?

A string is a sequence of characters enclosed in single (`' '`) or double (`" "`) quotation marks.

Examples:

```python
name = "Ali"
country = "Pakistan"
```

---

## 7. What is slicing?

Slicing is used to extract a portion of a sequence such as a string or list.

Example:

```python
text = "Python"

print(text[0:3])
```

Output:

```
Pyt
```

---

## 8. What is indexing?

Indexing is used to access a single element from a string or list using its position.

Example:

```python
text = "Python"

print(text[0])
```

Output:

```
P
```

---

## 9. What is dynamic typing?

Python is dynamically typed, meaning you do not need to specify the data type of a variable. Python automatically determines the type based on the assigned value.

Example:

```python
x = 10
x = "Hello"
```

The same variable can store different data types.
